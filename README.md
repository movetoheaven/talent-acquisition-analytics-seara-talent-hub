# Talent Acquisition Analytics for Seara Talent Hub
## Project Overview

Talent acquisition plays a critical role in ensuring sustainable business growth, especially for startups that rely on qualified talent to support rapid expansion. This project focuses on analyzing recruitment data for Seara Talent Hub, a fictional startup company, to gain insights into candidate demographics, skill availability, talent readiness, recruitment channel effectiveness, and candidate motivations.

Using Exploratory Data Analysis (EDA) and interactive dashboard development, this project aims to support data-driven talent acquisition strategies and help build a stronger talent pipeline for future workforce needs.
